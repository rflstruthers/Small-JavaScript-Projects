Coding the JavaScript:

The next step will be to write our JavaScript but there will be a few hurdles that we must traverse before it will be functional.

Since our script will ultimately present the user with a string of text that will list out the selected items and a running total, we will need to get all of the values from the first list of options: "Size".

The Size menu section will have four values:

  1. Personal Pizza
  2. Medium Pizza
  3. Large Pizza
  4. Extra Large Pizza
For our first step, we will need to begin our first function and declare and initialize our variables: function getReceipt() { var text1 = "<h3>You Ordered:</h3>; var runningTotal = 0; var sizeTotal = 0; var sizeArray = document.getElementsByClassName("size"); } So for these first few lines of code in our function getReceipt(), we declared several variables:
  1. var text1:
    This is a string that will show a list of selected items.

  2. var runningTotal:
    This is a numeric value and will be the running total for all the selected items.

  3. var sizeTotal:
    This is a numeric value and will be the subtotal of the selected pizza size.

  4. var sizeArray:
    This is an array of all the input options from the "Size" section.

What we will need to select from the Document Object Model(DOM) all of the input elements. We do this by leveraging a built-in JavaScript API method: document.getElementsByClassName("size");.

Since we have multiple input elements in the Size menu section, we will receive an array of all those input elements.

Next, we will need to iterate through the array with a For loop and then use an If statement to determine and print out each of the selected items from that array:

for (var i = 0; i < sizeArray.length; i++) { if (sizeArray[i].checked) { var selectedSize = sizeArray[i].value; text1 = text1+selectedSize+"<br>"; } }

Within the For loop, we will add in another If statement. The IF statement has a condition of (sizeArray[i].checked). This condition will check each item in the iteration to see if that particular item has been checked or not. Any items that our users' select will be denoted with the "checked" value.

In the IF statement, var selectedSize = sizeArray[i].value; we declared a new variable selectedSize. This variable will temporarily store the next selected item within our array so it can then be added to the text1 string variable.

The last line within the IF statement, (text1=text1+"Some new value"). Here we are concatenating a new value to the previous value already stored within text1. We concatenate the new value to the previous value, being careful not overwrite the previous value. This will effectively produce one long and continuous list of all selected items.

Example of the "behind-the-scene" view of this in action would look like the following example:

The original string "<h3>You Ordered:</h3>" gets concatenated with the new string "Large Pizza<br>" to make up the new string value: "<h3>You Ordered:</h3>Large Pizza<br>".

We ended our new string with the HTML, Line Break <br> tag. The Line break tag will force a new line within text1 ready for the next item to list out. This technique will effectively prevent us from accidently overwriting the previous stored value. Now, whenever there are additional selected items, we would then concatenate each item by using this same technique: "Original String" = ("Original String" + "A new string").

Example of the "behind-the-scene" view of this in action would look like the following example: "<h3>You Ordered:</h3>Large Pizza<br>Pepperoni<br>"

Next, we will need to reconcile the selected pizza size with its monetary value for our calculations. This next piece of code will look like:

if (selectedSize === "Personal Pizza") { sizeTotal = 6; } else if (selectedSize === "Medium Pizza") { sizeTotal = 10; } else if (selectedSize === "Large Pizza") { sizeTotal = 14; } else if (selectedSize === "Extra Large Pizza") { sizeTotal = 16; } This snippet of code is fairly straight forward but I will provide a brief explanation. Here we are using another If statement and several Else If statements, each using the condition (selectedSize === "A particular pizza"). Whenever the value stored within the variable selectedSize is a match with the condition, then the corresponding code, sizeTotal = number; will then get fired off, effectively storing the correct pizza value for the selected pizza size.

We will add the monetary value stored within sizeTotal to the runningTotal through the next snippet of code:

runningTotal = sizeTotal;

Here, the selected pizza subtotal will get added to the total amount, ensuring that our script will be fully aware of the total for the entire order.

Any instances of console.log() are for our own dev purposes and may be omitted from the final code.

The final line of code:

getMeat(runningTotal,text1);

This will call our next function in the series and pass along our variables, runningTotal and text1, ensuring that the next function will have access to their values contained and thus will be able to use or manipulate these values further.

Here is the entire code for the getReceipt() function:

function getReceipt() { // This initializes our string so it can get passed from // function to function, growing line by line into a full receipt var text1 = "<h3>You Ordered:</h3>"; var runningTotal = 0; var sizeTotal = 0; var sizeArray = document.getElementsByClassName("size"); for (var i = 0; i < sizeArray.length; i++) { if (sizeArray[i].checked) { var selectedSize = sizeArray[i].value; text1 = text1+selectedSize+"<br>"; } } if (selectedSize === "Personal Pizza") { sizeTotal = 6; } else if (selectedSize === "Medium Pizza") { sizeTotal = 10; } else if (selectedSize === "Large Pizza") { sizeTotal = 14; } else if (selectedSize === "Extra Large Pizza") { sizeTotal = 16; } runningTotal = sizeTotal; console.log(selectedSize+" = $"+sizeTotal+".00"); console.log("size text1: "+text1); console.log("subtotal: $"+runningTotal+".00"); getMeat(runningTotal,text1); // All three of these variables will be passed on to each function };
